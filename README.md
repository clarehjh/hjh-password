# 说明

修改密码组件支持vue2,vue3

## - 配置属性

| 字符             | 字段               | 默认值 |
| ---------------- | ------------------ | ------ |
| 是否显示修改密码 | isChangePassword | true   |
| 是否显示手机号   | isTelephone        | true   |
| 角色             | isole              | true   |
| 账号             | isUserName         | true   |

## - 配置方法

| 提交修改         | changePassword(val):void     | Function |
